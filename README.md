# Društvo GBS in CIDP Slovenije – GitHub Pages spletna stran

Ta mapa vsebuje pripravljeno statično spletno stran za objavo na GitHub Pages.

## Vsebina
- `index.html` – glavna stran
- `styles.css` – oblikovanje
- `script.js` – mobilni meni in drobne interakcije
- `assets/images/logo.png` – logotip društva
- `assets/docs/` – dokumenti za prenos

## Kratka objava na GitHub Pages
1. Ustvari nov GitHub repozitorij.
2. Naloži vse datoteke iz te mape v koren repozitorija.
3. V GitHub repozitoriju odpri **Settings → Pages**.
4. Kot vir objave izberi **Deploy from a branch**.
5. Izberi vejo **main** in mapo **/(root)**.
6. Shrani nastavitve in počakaj, da GitHub objavi stran.

## Dokumenti na strani
Na strani so že povezani:
- statut društva
- pristopna izjava za rednega člana
- pristopna izjava za podpornega člana
- donacijska pogodba
- sponzorska pogodba
- informativni dokument
- tridelna zloženka
- ZIP paket dokumentov

## Prilagoditev
- Za zamenjavo dokumenta ohrani isto ime datoteke v `assets/docs/`.
- Za spremembo kontaktnega e-naslova uredi vrstico `mailto:` v `index.html`.
- Za uporabo lastne domene jo nastavi v **Settings → Pages → Custom domain**.
