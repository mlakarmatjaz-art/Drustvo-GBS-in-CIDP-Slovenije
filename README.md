# Društvo GBS in CIDP Slovenije – spletna stran

Ta mapa vsebuje pripravljeno statično spletno stran za objavo na GitHub Pages.

## Vsebina
- `index.html` – glavna stran
- `styles.css` – slogovna datoteka
- `script.js` – mobilni meni in obrazec za članstvo
- `assets/` – logotipi in favicon
- `docs/` – dokumenti društva, ki so povezani na strani
- `.nojekyll` – priporočeno za GitHub Pages
- `POSTOPEK-OBJAVE.md` – navodila za objavo

## Kaj je že vključeno
- premium pregledna vstopna stran
- predstavitev društva in delovanja
- vsebina o GBS in CIDP
- povezave do dokumentov
- razdelek za članstvo
- razdelek za novice
- razdelek za podporo društvu
- kontaktni podatki

## Kako urejati
- novice urejaš v razdelku `#novice` v datoteki `index.html`
- kontaktni e-poštni naslov lahko zamenjaš na mestih, kjer je naveden `gbs.cidp.skupnost@gmail.com`
- ko boste imeli uradni TRR, ga dopiši v razdelek `#podpora`
